# Bioinfo Starter Guide

这是一个可直接通过 GitHub Pages 分享的静态网页版本。

在线访问：

- https://hi-fei-cool.github.io/bioinfo-starter-guide/
- https://github.com/hi-fei-cool/bioinfo-starter-guide

## 文件说明

- `index.html`: 网页主文件
- `R.png` / `Python.png`: R 与 Python 对比图片
- `仙桃学术.png`: 工具平台示意图
- `生信入门课.png` / `生信技能树.png`: 起步材料示意图

## 本地预览

在仓库目录运行：

```bash
python3 -m http.server 8000
```

然后打开 `http://127.0.0.1:8000/`。
